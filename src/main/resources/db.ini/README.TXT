此目录下用于存放数据库初始化脚本


1.基础资料：
舰船：舷号、种类（驱逐舰、护卫舰）(图标)、型号、名称、隶属单位、最大航速（节）,续航能力（海里）,满载排水量（吨），标准排水量（吨），生产研制单位、服役日期、修理情况、高清大图、主要武备；和雷弹的关系；

飞机：编号、种类（歼击机、运输机）(图标)、机型、隶属单位、最大航速（公里/时）,作战半径（公里）、最大起飞重量（吨）,生产研制单位、服役日期、修理情况、高清大图、主要武备；和雷弹的关系；

岸导:单位名称、种类(图标)、地址、经纬度、雷弹型号

仓储机构：名称、类别(图标)、地址、经纬度、人员、器材、雷弹

修理机构：名称、类别(图标)、地址、经纬度、人员、器材

雷弹：型号、生产研制单位、服役日期、技术状态（1,2,3级）、高清大图、性能、战备等级转进时间（小时）、装载时间（吊装时间）

人员：专业、数量、技术等级（技工、技师、高级技工）

器材：专业、种类、型号、数量
